for i in range(1,8):
    print("^"+str(i),end="\t")

    for j in range(1,8):
        print(i**j,end="\t")

## 判斷閏年
原始碼
```python
def year(y):
    if y % 4 == 0 and (y % 100 != 0 or y % 400 == 0):
        print("是閏年")
    else:
        print("不是閏年")

year(int(input("請輸入年份： ")))
```
輸出
```
請輸入年份： 2023
不是閏年

請輸入年份： 2020
是閏年

請輸入年份： 1234
不是閏年
```
